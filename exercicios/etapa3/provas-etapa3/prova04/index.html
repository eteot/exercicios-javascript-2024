Enunciado

A seguir, temos um exemplo de atividade para manipulação de elementos de um vetor. Crie as rotinas para:

  • Mostrar o vetor na tela. Considere a possibilidade dele estar vazio. [2,0 pontos]
  • Adicionar um novo elemento único ao final do vetor. Não poderemos ter dois elementos iguais em um vetor [3,0 pontos]
  • Mostrar quantos elementos ao todo temos no vetor [1,0 ponto]
